2

4

🗓️ Connect Google Calendar

You need a Google OAuth Client ID. Here's how to get one in ~2 min:

1

Open console.cloud.google.com → your project → APIs & Services → Library → search Google Calendar API → Enable.

2

Go to Credentials → Create Credentials → OAuth 2.0 Client ID. Type: Web application. Add https://cadies-corner.netlify.app as an Authorized JavaScript origin.

3

Copy the Client ID and paste it below.

✉️ Connect Gmail

Uses the same Google OAuth Client ID as Calendar. If you already set that up, paste the same Client ID below.

1

Open console.cloud.google.com → your project → APIs & Services → Library → search Gmail API → Enable.

2

Make sure http://localhost:3000 is added as an Authorized JavaScript origin under your OAuth Client ID credentials.

3

Paste your Client ID below (same one used for Calendar is fine).
